# Overview

As a software engineer continuously working to expand my technical skill set,
I challenged myself to strengthen my JavaScript and Node.js skills by building
a practical, real-world application from scratch. My goal was to gain hands-on
experience with file handling, user input, data processing, and modern JavaScript
features such as ES6 array methods.

The software I built is a Personal Finance Tracker — a command-line application
written in JavaScript (Node.js) that allows users to record income and expenses,
view transaction history, and calculate their financial balance. The application
stores all data in a JSON file, ensuring that financial records persist between sessions.

My purpose for writing this software was to move beyond simple coding exercises
and build a functional tool that integrates multiple core programming concepts.
This project helped me practice working with modules, file I/O, arrays, user input,
and error handling in a cohesive and meaningful way.

[Software Demo Video](http://youtube.link.goes.here)

# Development Environment

- **IDE:** Visual Studio Code
- **Runtime:** Node.js
- **Version Control:** Git / GitHub

The project is written in **JavaScript** using **Node.js**. It utilizes built-in
Node modules such as `fs` for file handling, `path` for file paths, and
`readline` for user interaction in the command-line interface.

Additionally, external libraries such as `chalk` (for styling terminal output)
may be used to improve user experience.
